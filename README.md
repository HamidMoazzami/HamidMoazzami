## About Me  
Applied mathematician and machine learning researcher working at the intersection of PDE-constrained optimization, neural operators, adaptive wavelet methods, and data assimilation. Developing scalable numerical algorithms and operator-learning frameworks for nonlinear dynamical systems, inverse problems, and adjoint-based optimization.


---

## 🧩 Selected Projects 
🔹 Meta-learning with Fourier Neural Operators for PDE-constrained optimization problem

🔹 Adaptive wavelet conjugate gradient and Gauss--Newton methods for nonlinear PDEs

🔹 Wavelet multigrid methods for solving Hessian systems in variational data assimilation

🔹 Pseudo-spectral methods for solving chaotic PDEs in sequential quadratic programming  

---

## ⚙️ Technical Skills  
**Languages:** Python, R, MATLAB, C/C++

**Core Expertise**

- **Machine Learning:** Operator Learning, Deep Learning for Dynamical Systems, and Scalable Predictive Modeling.
- **Statistical & Signal Processing:** Time-Series Analysis (ARMA, ARIMA, Local Trend Models, etc.), Probabilistic State Estimation (Kalman Filtering), Statistical Inference, Feature Selection (AIC, BIC, and p-values), Nonparametric Smoothing, and Fourier- and Wavelet-Based Multiscale Analysis.
